# Python 模組(Module)
```
#!/usr/bin/python
# -*- coding: UTF-8 -*-
 
# 導|載入模組
import support
 
# 現在可以調用模組裡包含的函數了
support.print_func("MyFriends")
```
## support.py
```
def print_func( par ):
   print("Hello : ", par)
   return
```

# Python套件(package|包)
```
#!/usr/bin/python
# -*- coding: UTF-8 -*-
 
# 導|載入模組
import my_module
 
# 現在可以調用模組裡包含的函數了
support.print_func("MyFriends")
```
